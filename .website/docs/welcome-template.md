# 欢迎使用华南农业大学学习资料库

&nbsp;&nbsp;&nbsp;&nbsp;本系统为 **Web 开发设计课程** 项目成果，用于集中展示与管理课程相关的学习资料与试卷内容，提供清晰的分类导航与便捷的资料访问方式。

---

## 📚 项目简介

&nbsp;&nbsp;&nbsp;&nbsp;- 本项目采用 **HTML、CSS 与 JavaScript** 构建前端页面
&nbsp;&nbsp;&nbsp;&nbsp;- 采用rust作为后端，使用tauri架构
&nbsp;&nbsp;&nbsp;&nbsp;- 通过统一的导航结构组织不同课程与资料类型
&nbsp;&nbsp;&nbsp;&nbsp;- 采用 Markdown 格式文档作为主要内容文本，动态加载与展示

---

## 🧭 使用说明

&nbsp;&nbsp;&nbsp;&nbsp;1. 请通过左侧导航栏选择需要查看的课程或资料
&nbsp;&nbsp;&nbsp;&nbsp;2. 点击具体条目即可在右侧区域加载对应内容
&nbsp;&nbsp;&nbsp;&nbsp;3. 点击左上角标题可随时返回本欢迎页面
&nbsp;&nbsp;&nbsp;&nbsp;4. 若所选资料不存在，将自动显示错误提示页面

---

## 🔍 导航搜索

&nbsp;&nbsp;&nbsp;&nbsp;- 侧边栏顶部提供关键词搜索框
&nbsp;&nbsp;&nbsp;&nbsp;- 可对 **课程分类名称** 与 **具体科目名称** 进行快速检索
&nbsp;&nbsp;&nbsp;&nbsp;- 搜索效果类似于浏览器的查找功能（Ctrl + F），但仅作用于导航区域
